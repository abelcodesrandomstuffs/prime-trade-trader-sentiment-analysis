# PrimeTrade AI Hiring Assignment

## Objective

This project analyzes the relationship between Bitcoin Market Sentiment (Fear & Greed Index) and Hyperliquid trader performance.

---

## Repository Contents

- analysis_script.py
- charts_script.py
- historical_data.csv
- fear_greed_index.csv
- Trader_Sentiment_Analysis_Report.pdf

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib

---

## Key Findings

- Trading activity increases during Fear.
- Traders use larger position sizes during Fear.
- Traders become more Short during Greed.
- Win rate is highest during Extreme Greed and Fear.
- Different cryptocurrencies react differently to market sentiment.

---

## How to Run

```bash
python3 analysis_script.py
python3 charts_script.py
```
